# AI Text Refiner – Chrome Extension (Groq)

## Overview
`AI Text Refiner` is a lightweight Chrome Extension (Manifest V3) that lets you instantly improve or re‑format any text you are writing in the browser. It works on any `<textarea>` or `contenteditable` element and offers three actions:

- **✨ Default** – generic refinement (clean up grammar, style, brevity). 
- **📧 Email** – expands the content into a professional email. 
- **📱 Social** – turns the content into a punchy social‑media post with emojis and hashtags.

The heavy lifting is done by **Groq's Llama 3.1‑70b‑versatile** model (fast, OpenAI‑compatible API).

## Installation
1. Open Chrome and go to `chrome://extensions`.
2. Enable **Developer mode** (top‑right toggle).
3. Click **Load unpacked** and select the folder:
   ```
   E:\Projects\Refine
   ```
4. The extension icon should appear next to the address bar.

---

## First‑time setup – add your Groq API key
1. Click the extension icon → a popup appears.
2. Paste your **Groq API key** (the same key you use for `https://api.groq.com`).
3. Press **Save**.
4. The status line will confirm `✅ Groq API key saved.`

> The key is stored locally via `chrome.storage.local` under the name `groq_key` – it never leaves your browser.

---

## How to use the extension
1. Focus a text field (e.g., a Gmail compose box, a comment box, a simple textarea on any website).
2. Three small buttons appear on the right side of the field:
   - **✨** – generic refinement.
   - **📧** – refine *as an email*.
   - **📱** – refine *as a social post*.
3. Click the desired button.
   - While waiting, the button shows a loading spinner (`⏳`).
   - When the AI response arrives, the original text is replaced with the refined version.
4. If something goes wrong you will see an alert with the error message.

### Example workflow
- **Drafting a quick note:** Write a rough outline, hit **✨**, and the text is cleaned up.
- **Turning notes into an email:** Write bullet points, click **📧**, and you get a ready‑to‑send email.
- **Posting on social:** Write a few keywords, hit **📱**, and get a post with emojis and hashtags.

## Changing the model (optional)

```js
// Current model
model: 'llama-3.1-70b-versatile',
```
Replace it with:
```js
model: 'llama-3.1-8b-instant',
```
Save the file and reload the extension.

## Development notes
- **Popup (`popup.html` / `popup.js`)** – stores the API key.
- **Content script (`content.js`)** – injects the three buttons and sends a message `{action: 'refineText', text, format}` to the background.
- **Background (`background.js`)** – fetches the key, builds the appropriate system prompt (default, email, social), calls Groq, and returns `refined`.
- The extension does **not** collect any user data beyond what you type into the field; everything stays local.

## FAQ

  **A:** Yes – you need a valid API key. Free tiers are fine for light usage.
- **Q:** *Can I use OpenAI instead?*  
  **A:** The code is modular; just replace the fetch URL and model name in `background.js` and store the key under `openaiKey`.
